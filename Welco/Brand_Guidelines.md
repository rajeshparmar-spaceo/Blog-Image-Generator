# Brand Guidelines

## 1. Background & Color Palette

### Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Primary Blue** | `#004CE6` | Core brand color, CTAs, key accents, circle frames |
| **Secondary Light Blue** | `#EBF1FF` | AI response chat bubbles, highlights |

### Background Styles

| Style | Usage | Description |
|-------|-------|-------------|
| **Solid Light Blue** (~75% of images) | Default for most Type A layouts | Flat very light blue (`#E4ECF7` / `#E8F0FE`) background, sometimes with very faint grid lines at the bottom |
| **Soft Gradient** (~25% of images) | Gradient variant for Type A & Type B | Light blue `#B2E8FF` to medium blue `#80AAFF` to soft pink/purple `#F0B2FF`, typically 135deg or left-to-right |

### Gradient Colors (when used)

| Color | Hex Code |
|-------|----------|
| Light Blue | `#B2E8FF` |
| Medium Blue | `#80AAFF` |
| Soft Purple | `#F0B2FF` |

### Text Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Black** | `#000000` | Headlines, primary text, chat bubble text |
| **Light Blue** | `#EBF1FF` | Highlights and emphasis (rare) |

## 2. Typography & Text Elements

### Font Family

- **Primary Typeface:** Inter (Sans-serif)

### Font Sizes

| Element | Size |
|---------|------|
| Headings | 28-30px |
| Body Text / Chat bubbles | 14-16px (Regular) |

> Minor size adjustments are allowed if needed.

### Font Weight Hierarchy

| Element | Weight |
|---------|--------|
| Headlines | Semi-Bold (600) |
| Subheadings | Medium (500-600) |
| Body Text | Regular (400) |

### Text Treatment

- Keep text overlays minimal (5-10 words max)
- Maintain high contrast (dark text on light backgrounds)
- Avoid script/decorative fonts -- stay professional and readable

## 3. Visual Style & Composition

### Iconography

- Outline icons only
- Stroke width: 2px
- Clean, consistent, and minimal

### Imagery Style

- Use realistic, high-quality photos AND AI-generated realistic renders
- Avoid overly staged or generic stock visuals

### Illustration Style

- **3D isometric elements** -- Modern, tech-forward aesthetic
- **Flat design with depth** -- Clean illustrations with subtle shadows
- **Abstract shapes** -- Circles, rounded rectangles, organic shapes
- **Tech iconography** -- Laptops, phones, documents, AI symbols

### Layout & Composition Rules

#### Canvas Dimensions

| Property | Value |
|----------|-------|
| **Standard size** | 800px x 600px |
| **Aspect ratio** | 4:3 (Landscape) |
| **Format** | Landscape, always |

#### Primary Layout: Centered Composition (Used in 100% of reference images)

Unlike other brands in this project, Welco uses a **centered composition** -- NOT a left-text / right-visual split.

| Zone | Position | Content |
|------|----------|---------|
| **Top Headline Zone** | Top 15-20% of canvas, centered | Headline text (2 lines max, centered) |
| **Central Visual Zone** | Middle 60-70% of canvas | Receptionist photos, chat conversation bubbles, accent icons, caller photos |
| **Bottom Logo Zone** | Bottom-right corner | welco.ai text logo |

#### Layout Variants

| Variant | Trigger / Use Case | Description |
|---------|-------------------|-------------|
| **Type A -- Receptionist + Chat Conversation** (~75% of images) | Pricing guides, how-to articles, feature explainers | Centered headline top; receptionist photo (varied positioning) with headset; chat conversation bubbles showing customer-AI dialog; optional caller photo; accent icons (dollar, 24/7, calendar) |
| **Type B -- Listicle / Comparison** (~25% of images) | "Best X Services", "Top Alternatives" posts | Centered headline top; horizontal row of overlapping circles with company logos OR scattered grid of rounded-square app icons; welco headset icon is largest and central; no receptionist photo |

#### Type A Sub-Variants (Observed in reference images)

| Sub-Variant | Examples | Key Details |
|-------------|----------|-------------|
| **A1 -- Receptionist alone + Chat** | Images 5, 6 | Receptionist center/center-left, chat bubbles on right, NO caller photo. May include accent icons (24/7, calendar). |
| **A2 -- Receptionist + Caller + Chat** | Images 2, 3, 4, 7 | Receptionist on one side, caller on the other, chat bubbles between them. Caller can be LEFT or RIGHT. |
| **A3 -- Receptionist with props (tablet/laptop)** | Images 3, 7 | AI-illustrated receptionist holding a tablet or near a laptop/screen mockup. Adds a workspace element. Usually paired with caller photo and "AI Receptionists" label badge. |

#### Text Placement Rules

| Property | Specification |
|----------|---------------|
| Headline alignment | **ALWAYS centered horizontally** at top of canvas |
| Headline lines | Maximum 2 lines |
| Headline vertical position | Top 15-20% of canvas, ~40-60px from top |
| Headline style | Dark black (#000000), Inter font, ~28-30px, weight 600 |
| Subtitle | Rarely used -- most images have headline only, no subtitle |
| Text max-width | ~90% of canvas width (centered) |
| Padding top | 40-60px |

#### Chat Conversation UI (Distinctive Welco Element)

The chat conversation is the **primary visual storytelling element** -- it demonstrates the AI receptionist in action.

| Property | Specification |
|----------|---------------|
| **Placement** | Center or center-right of canvas, between receptionist and caller (or beside receptionist if no caller) |
| **Message count** | 2-4 alternating exchanges (customer - AI - customer - AI); most images show 3-4, minimum observed is 2 |
| **Customer bubbles** | White/off-white background, black text, rounded rectangle (~12px radius) |
| **AI response bubbles** | Light blue/lavender tint (#EBF1FF / #C5D5F6), black text, rounded rectangle (~12px radius) |
| **Font** | Inter, 14-15px, regular weight |
| **Content** | Realistic appointment booking / service request conversations (book appointment, confirm time, send confirmation, request estimate) |
| **Shadow** | Very subtle drop shadow on bubbles |
| **Bubble max-width** | ~200-250px per bubble |

#### Receptionist Photography Rules (Type A layouts)

| Property | Specification |
|----------|---------------|
| **Subject** | Smiling professional woman with headset (AI receptionist persona). MULTIPLE different women are used across images -- variety is expected (different hair colors, styles, ethnicities). |
| **Expression** | Friendly, approachable, smiling |
| **Attire** | Professional -- blazer, business casual |
| **Equipment** | Over-ear headset with microphone (always present) |
| **Positioning** | Varies: center, center-left, or center-right of canvas. Shoulders up or upper body. |
| **Photo style** | Mix of realistic photography AND AI-generated realistic renders |
| **Frame options** | (a) No frame -- photo directly on background (~50% of Type A), (b) Circular frame with blue (#004CE6) border/background (~40%), (c) Within a screen/device mockup (~10%) |
| **Props** | Sometimes holding tablet or near laptop/screen (AI-illustrated variants) |

#### Caller Photo Rules (Type A layouts -- optional)

| Property | Specification |
|----------|---------------|
| **Presence** | Optional -- present in ~67% of Type A images |
| **Subject** | Person on phone (customer), casual or professional setting |
| **Positioning** | Flexible -- can be on LEFT or RIGHT side, opposite the receptionist |
| **Frame options** | Circular frame (gray/light border), rectangular photo, or small square with rounded corners |
| **Size** | Varies: small thumbnail (~60-80px) to large photo (~200px+) |
| **Photo style** | Realistic photography |

#### Accent & Decorative Elements

| Element | Usage | Style |
|---------|-------|-------|
| **Dollar icon circle** | Pricing/cost articles | Blue (#004CE6) circle with white dollar sign, ~50-60px, top-left or top-right |
| **24/7 icon** | Automation/after-hours articles | Blue circle with "24/7" text and circular dashed arrows, ~50-60px |
| **Calendar icon** | Appointment booking articles | Blue circle with calendar symbol and circular arrows |
| **Green status dot** | Near receptionist photo | Solid green (#4CAF50) circle, ~15-40px (size varies), indicates "online" |
| **Headset icon (small)** | Decorative in Type A layouts | Small welco headset symbol near chat bubbles, ~20-30px |
| **Headset icon (large)** | Listicle/comparison layouts (Type B) | Welco brand headset symbol in large rounded square, central placement |
| **"AI Receptionists" label** | Near receptionist photo (especially A3 variant) | Dark text in white/light rounded rectangle badge |
| **Subtle grid pattern** | Background texture at bottom | Very faint grid lines in light gray, barely visible, used in ~40% of images |

#### Listicle / Comparison Layout Rules (Type B)

| Property | Specification |
|----------|---------------|
| **Style 1: Overlapping circles** | Horizontal row of overlapping circles with company logos, welco headset icon largest in center with blue (#004CE6) outline |
| **Style 2: Scattered app icons** | Organic/scattered grid of rounded-square app icons (16-20px radius, subtle shadow), welco icon largest and central. Small curved arrow from competitor icon pointing to welco icon. |
| **Circle size (Style 1)** | Central: ~150px diameter; others: ~100-120px, partially overlapping |
| **App icon size (Style 2)** | Central: ~100-120px square; others: ~50-70px square, scattered organically around center |
| **Background** | Solid light blue OR light blue to very soft pink gradient |

#### Spacing & Proportion Rules

- Use generous white space -- don't overcrowd
- Headline should have ~40px clear space below before visual elements start
- Chat bubbles: 8-12px vertical gap between messages
- Maintain ~25px minimum margin from all canvas edges
- Receptionist photo should not touch canvas edges
- Accent icons positioned to balance composition (typically opposite side from chat bubbles)

## 4. Imagery & Graphics

### Core Visual Elements

- Receptionist photos with headsets (primary visual element)
- Chat conversation UI showing AI-customer dialog
- Company/service logo icons in circles or rounded squares (listicle layouts)
- Blue accent icons (dollar, calendar, 24/7, headset)

### Photography

- **Receptionist imagery is central** -- friendly, professional women with headsets
- **Multiple different receptionist personas** are used -- variety in appearance is expected
- **Caller imagery** -- people on phones, casual/home/office settings
- Photos placed in circular frames with blue borders OR directly on background
- Clean, bright, professional look -- no dark or moody lighting
- **AI-generated realistic renders** acceptable and commonly used for receptionist persona

### What to Avoid

- No generic stock photos without context
- No workspace/desk shots (unlike other brands)
- No coding/developer imagery
- No complex dashboard mockups
- No illustration-heavy compositions

## 6. Logo Placement Guide

### Logo File

| Property | Value |
|----------|-------|
| **File** | `Welco Logo Dark Transparent.png` |
| **Source size** | 636px x 120px |
| **Contents** | "welco.ai" lowercase dark text mark -- the 'o' in welco has a distinctive headset/circle design |

### Placement & Sizing

| Property | Value |
|----------|-------|
| **Position** | ALWAYS bottom-right corner |
| **Display width** | 85-90px |
| **Display height** | ~25px |
| **Margin from right** | 25px |
| **Margin from bottom** | 25px |
| **Background badge** | None (logo placed directly on image) |
| **Visibility** | Present in ~87% of images (omitted in some listicle/comparison layouts where welco headset icon serves as brand identifier) |

### Placement Rules

- Logo should always appear in the **bottom-right** corner
- Use the dark version on light/gradient backgrounds
- Keep logo small and subtle -- it should not compete with headline text
- Maintain at least 25px clear space around the logo
- Do not place the logo over busy or dark image areas
- In listicle layouts where the welco headset icon is prominently displayed as the central element, the text logo may be omitted

## 7. Brand Consistency

- **Design principles:** Clean, modern, functional, AI-focused
- **Welco color palette:**
  - Primary: `#004CE6`
  - Secondary (Light Blue): `#EBF1FF`
  - Default Background: `#E4ECF7` / `#E8F0FE` (solid light blue)
  - Soft Background Gradient (variant): `#B2E8FF`, `#80AAFF`, `#F0B2FF`
- **Accessibility:** WCAG 2.1 AA compliant contrast ratios (4.5:1 minimum)

---

## 8. Common Mistakes to Avoid

| Avoid |
|-------|
| Cluttered compositions with too many elements |
| Low contrast text that's hard to read |
| Overly complex gradients or patterns |
| Generic stock photos without context |
| Inconsistent styling across image series |
| Using only one receptionist persona -- variety is part of the brand |
| Placing caller always on same side -- positioning should vary |
| Using left-right split layout instead of centered composition |
