## 2. Typography & Text Treatment

### Font Family

* **Primary Headline Typeface:** New Kansas
* **Secondary Typeface (Subtitles, UI & Body):** Poppins

---

### Font Sizes

| Element            | Size    |
| ------------------ | ------- |
| Main Headline (H1) | 56–64px |
| Subtitle           | 22–26px |
| Section Heading    | 30px    |
| Body Text          | 16px    |
| UI Card Text       | 14–16px |
| Micro Badge Text   | 12px    |

> Minor size adjustments are allowed for responsiveness.

---

### Font Weight Hierarchy (Poppins)

| Element     | Weight        |
| ----------- | ------------- |
| Subheadings | Medium (600)  |
| Body Text   | Regular (400) |
| UI Labels   | Medium (500)  |

---

### Headline Styling Rules (New Kansas)

* Use for main blog titles only
* Sentence case preferred
* Tight but elegant line height
* Avoid excessive tracking
* Maximum 2–3 lines for banner titles

---

### Text Usage Guidelines

* Keep banner text concise (5–10 words ideal for main line)
* Maintain strong contrast against lavender background
* Avoid decorative or script fonts
* No mixed font families beyond New Kansas + Poppins
* Maintain consistent spacing between title and subtitle

---

## 3. Visual Style & Composition

### Iconography

* Outline-style icons only
* Stroke width: 1.5–2px
* Clean, minimal, and consistent
* Use database, cloud, workflow, and system-related icons

---

### Imagery Style

* Real CRM UI screenshots
* Rounded corners (16–20px radius)
* Subtle drop shadow
* Slight background blur when layering UI

Avoid:

* Generic stock photography
* Human lifestyle imagery
* Overly dramatic lighting effects

---

### Illustration Style

* Soft UI cards with light purple fill
* Circular frames for platform logos
* Subtle directional arrows (left → right flow)
* Minimal 3D infrastructure visuals (cloud, server, database)
* Clean SaaS dashboard aesthetic

---

### Layout & Composition Rules

| Property                 | Specification           |
| ------------------------ | ----------------------- |
| Preferred format         | Landscape               |
| Blog Featured Image Size | **1365px × 640px**      |
| Structure                | Left text, right visual |
| Border radius            | 16–20px for UI elements |

Layout Pattern:

* Headline (New Kansas) on left
* Subtitle (Poppins) below headline
* Brand logo bottom-left (primary) or bottom-center with white pill badge (centered layouts)
* Platform visuals / UI cards on right
* Directional arrows to indicate migration or flow

Design Principles:

* Generous whitespace
* Clear visual hierarchy
* Rule of thirds alignment
* No visual clutter

---

## 4. Imagery & Graphic Elements

### Core Visual Elements

* CRM platform logos inside white circular containers
* Migration arrows
* Feature checklist cards (Contacts, Accounts, Workflows, Reports)
* Cloud and infrastructure icons (AWS, self-hosted indicators)
* Dashboard previews and structured UI blocks

---

### Photography Guidelines

* No real-world photography
* Focus on systems, dashboards, architecture

## 7. Brand Consistency & Accessibility

* Maintain a clean, SaaS-native aesthetic
* Use subtle gradients — avoid heavy saturation
* Ensure WCAG 2.1 AA contrast ratio (minimum 4.5:1)
* Avoid low-contrast purple-on-purple text
* Maintain consistent typography usage (New Kansas + Poppins only)

---

## 8. Common Mistakes to Avoid

| Avoid                                      |
| ------------------------------------------ |
| Overcrowded compositions                   |
| Large paragraphs inside banners            |
| Heavy shadows or glow effects              |
| Inconsistent arrow styles                  |
| Mixing additional font families            |
| Using stock photos or human portraits      |
| Poor contrast between title and background |
